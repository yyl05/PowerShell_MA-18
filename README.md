# Projet du module MA‑18 (CPNV 2025/2026)

- **Objectif** : Approfondir les compétences PowerShell à travers deux thèmes distincts, avec création d'exercices, résumés, scripts et séquence d’apprentissage.


## Présentation du projet
Ce projet s’inscrit dans le cadre du module MA‑18 et se déroule sur 8 semaines, réparties en deux thèmes :

- **Administration réseau** (semaines 1 à 4)
- **Administration serveur** (semaines 5 à 8)


# Administration réseau

Découvrir et utiliser les cmdlets réseau PowerShell

Diagnostiquer un réseau

Manipuler les cartes réseau, les adresses IP, le DNS et le firewall
